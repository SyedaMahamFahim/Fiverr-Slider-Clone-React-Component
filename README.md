# Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Fiverr-Slider-Clone-React-Component.git
```

Go to the project directory 

```bash
  cd fiverr-slider-clone-react-component
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm start
```
